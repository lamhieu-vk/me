# usage

### clone

```bash
$ git clone https://github.com/lamhieu-vk/me.git
```

### setup

```bash
$ npm install
```

### review

```bash
$ npm run start
```

### build

```bash
$ npm run build
```

### deploy

```bash
$ npm run deploy
```
